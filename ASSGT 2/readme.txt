# My Creative Website

This is a simple website created for an assignment. It includes various HTML and HTML5 controls, an external CSS file, and adheres to the provided guidelines.

## HTML and HTML5 Controls Used
- Favicon (`<link rel="icon">`)
- Table (`<table>`)
- Form (Not included in the sample, but you can add it as needed)
- Images (`<img>`)
- Hyperlink (`<a>`)
- Button (`<button>`)
- Audio (`<audio>`)
- Video (`<video>`)
- Header (`<header>`)
- Footer (`<footer>`)
- Summary (`<summary>`)
- Menu (Implemented as a navigation menu using `<nav>` and `<ul>`)
- Contact Information (`<tel>` for phone, `<mailto>` for email)

## CSS
An external CSS file (`styles.css`) is used to style the website. It defines styles for various elements like the header, sections, tables, and footer.

## Colors and Layout
The website uses a simple color scheme with a white background, dark header, and black text. It is divided into sections for better visual appeal.
